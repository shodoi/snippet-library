# スニペット帳 (Snippet Library)

よく使うターミナルコマンドや、AIエージェント向け指示（プロンプト）をまとめて管理・コピーできる、シンプルでモダンなシングルページアプリケーション (SPA) です。

## 特徴

- ⚡ **軽快なSPA**: 単一の HTML ファイルとしてポータブルに動作します。
- ♿ **アクセシブル設計 (WCAG/WAI-ARIA準拠)**:
  - 4.5:1 以上の十分なカラーコントラスト比の確保
  - キーボード操作時の明瞭なフォーカスリング（`:focus-visible`）
  - スクリーンリーダー向け動的通知（アナウンサー）および適切な `aria-*` 属性の設定
  - キーボードユーザーに配慮したフォーカス管理（フォーム開閉時のフォーカス追跡と復元）
- 🛠️ **高いユーザビリティ**:
  - `<form>` セマンティクスによる Enter キー送信とブラウザ標準バリデーション
  - `Esc` キーによる入力キャンセルなどのキーボードショートカット
- 🔒 **セキュリティ＆堅牢性**:
  - `crypto.randomUUID` のフォールバックによる古い環境への互換性
  - ローカルストレージデータの構造バリデーションによるプロトタイプ汚染やクラッシュ防止
- 💾 **透過的なストレージ管理**:
  - `window.storage` API と `localStorage` を自動的に切り替える非同期ストレージラッパーを内蔵

## 動作方法

### 1. ブラウザで直接開く
`index.html` ファイルを Chrome、Edge、Safari などのモダンブラウザにドラッグ＆ドロップするか、ダブルクリックするだけで、特別な設定なしでそのまま動作します。

### 2. ローカル HTTP サーバーで起動する
Node.js や Python などがインストールされている場合、ローカルサーバーを起動して実行することも可能です。

**Node.js を使用する場合 (npx):**
```bash
npx http-server -p 8999
```

**Python を使用する場合:**
```bash
python -m http.server 8999
```

起動後、ブラウザで `http://localhost:8999` にアクセスしてください。

**サーバーの停止:**
起動したターミナル上で `Ctrl + C` をキーボードで押すことで、サーバーを停止できます。

## GitHub Gist 同期機能

スニペットデータを GitHub Gist に保存（バックアップ）し、異なるブラウザや端末から読み込み（復元）を行うことができます。

### 設定方法

1. **GitHub Personal Access Token (PAT) の取得**
   - GitHub にログインし、`gists` 権限（スコープ）を付与したトークンを作成します。
2. **接続設定の入力**
   - アプリ画面右上にある「設定」ボタンをクリックしてデータ管理設定ダイアログを開きます。
   - 作成したトークンを **GitHub Personal Access Token** 入力欄に入力します（パスワード表示トグルボタンで表示/非表示を切り替えられます）。
   - トークンの保存方法（「セッション限り (推奨)」または「ブラウザに永続保存」）を選択します。
   - **Gist ID**:
     - 初めて同期する場合は、**Gist ID を空欄のまま**にしておいてください。保存時に自動的に新規の Secret Gist が作成され、ID が自動入力されます。
     - すでにバックアップ済みの Gist がある場合は、その **Gist ID** を入力します。

### 同期操作

- **Gistへ保存（バックアップ）**
  - 「Gistへ保存」ボタンをクリックすると、現在のローカルスニペットが指定された Gist に上書き保存されます。
  - 成功すると「最終保存」の日時が更新されます。
- **Gistから読み込み（復元）**
  - 「Gistから読み込み」ボタンをクリックすると、Gist から `snippets.json` を取得してローカルデータを上書き復元します。
  - 上書きする前に確認ダイアログが表示されます。
  - 成功すると「最終読込」の日時が更新され、一覧が再描画されます。

> [!WARNING]
> **セキュリティに関する注意点**
> Gist は非公開（Secret）として作成されますが、URL を知っている第三者はアクセス可能です。スニペットの中に API キー、パスワード、暗号キーなどの機密情報を記述して同期しないよう十分に注意してください。
