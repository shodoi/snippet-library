# スニペット帳 (Snippet Library)

よく使うターミナルコマンドや、AIエージェント向け指示（プロンプト）をまとめて管理・コピーできる、シンプルでモダンなシングルページアプリケーション (SPA) です。

## 特徴

- ⚡ **軽快なSPA**: 単一の HTML ファイルとしてポータブルに動作します。
- ♿ **アクセシブル設計 (WCAG/WAI-ARIA準拠)**:
  - 4.5:1 以上の十分なカラーコントラスト比の確保
  - キーボード操作時の明瞭なフォーカスリング（`:focus-visible`）
  - スクリーンリーダー向け動的通知（アナウンサー）および適切な `aria-*` 属性の設定
  - キーボードユーザーに配慮したフォーカス管理（フォーム開閉時のフォーカス追跡と復元）
- 🛠️ **高いユーザビリティ**:
  - `<form>` セマンティクスによる Enter キー送信とブラウザ標準バリデーション
  - `Esc` キーによる入力キャンセルなどのキーボードショートカット
- 🔒 **セキュリティ＆堅牢性**:
  - `crypto.randomUUID` のフォールバックによる古い環境への互換性
  - ローカルストレージデータの構造バリデーションによるプロトタイプ汚染やクラッシュ防止
- 💾 **透過的なストレージ管理**:
  - `window.storage` API と `localStorage` を自動的に切り替える非同期ストレージラッパーを内蔵

## 動作方法

### 1. ブラウザで直接開く
`index.html` ファイルを Chrome、Edge、Safari などのモダンブラウザにドラッグ＆ドロップするか、ダブルクリックするだけで、特別な設定なしでそのまま動作します。

### 2. ローカル HTTP サーバーで起動する
Node.js や Python などがインストールされている場合、ローカルサーバーを起動して実行することも可能です。

**Node.js を使用する場合 (npx):**
```bash
npx http-server -p 8999
```

**Python を使用する場合:**
```bash
python -m http.server 8999
```

起動後、ブラウザで `http://localhost:8999` にアクセスしてください。

**サーバーの停止:**
起動したターミナル上で `Ctrl + C` をキーボードで押すことで、サーバーを停止できます。
